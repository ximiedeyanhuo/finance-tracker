## 🏗️ 技术架构

### 架构模式
- **MVVM + Clean Architecture**
- 分层清晰：UI → ViewModel → Domain → Data

### 核心技术栈
| 组件 | 版本 | 用途 |
|------|------|------|
| Kotlin | 1.9.20 | 开发语言 |
| Jetpack Compose | 1.5.4 | UI 框架 |
| Room | 2.6.1 | 数据库 |
| Hilt | 2.48.1 | 依赖注入 |
| Flow | 1.7.3 | 响应式编程 |
| Coil | 2.5.0 | 图片加载 |
| OpenCSV | 5.9 | CSV 导出 |

### 项目结构
```
finance-tracker/
├── data/           # 数据层（Entity、DAO、Repository）
├── domain/         # 领域层（Model、Service）
├── ui/             # UI 层（Compose、ViewModel）
│   ├── components/ # 通用组件
│   ├── screens/    # 页面组件
│   ├── theme/      # 主题系统
│   └── viewmodel/  # ViewModel
└── di/             # 依赖注入模块
```

## 🚀 快速开始

### 方式一：Android Studio（推荐）

```
1. 打开 Android Studio
2. File → Open → /workspace/finance-tracker
3. 等待 Gradle 同步完成
4. 点击 Run 按钮运行到模拟器/真机
```

### 方式二：命令行

```bash
cd /workspace/finance-tracker

# 清理并编译
./gradlew clean assembleDebug

# APK 位置
app/build/outputs/apk/debug/app-debug.apk

# 安装到设备
adb install app/build/outputs/apk/debug/app-debug.apk
```

### 编译要求
- Android Studio Hedgehog (2023.1.1) 或更高
- JDK 17
- Android SDK API 34
- 最低支持 Android 8.0 (API 26)
